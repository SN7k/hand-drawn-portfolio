## Features

- **Sketch-Style UI** - Hand-drawn borders, paper textures, playful animations
- **Dynamic Themes** - Interactive hover effects change page background colors
- **Mobile-First** - Responsive design with browser back/forward navigation
- **Fast & Modern** - Vite-powered with TypeScript and Tailwind CSS
- **Contact Form** - Built-in mailto integration with validation

## Quick Start

```bash
git clone https://github.com/SN7k/portfolio.git
cd portfolio
npm install
npm run dev
```

## Customization

Edit `src/data/portfolio.json` to update your personal info, projects, tech stack, and social links. All content is centralized for easy management.
